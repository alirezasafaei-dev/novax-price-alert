# شروع سریع - بات نواکس

## 🚀 بات آماده است!

بات نواکس الان در تلگرام فعال و آماده استفاده است.

### دسترسی به بات

در تلگرام به آدرس زیر بروید:

```
@novax_price_bot
```

یا از لینک زیر استفاده کنید:

```
https://t.me/novax_price_bot
```

## ⚡ شروع سریع (منومحور)

بات کاملاً دکمه‌ای است؛ فقط `/start` را بزنید و بقیه را از روی دکمه‌ها انجام دهید.

### دریافت قیمت‌ها

💰 قیمت‌ها ← بازار را انتخاب کنید:
- **₿ کریپتو**: BTC / ETH / SOL / BNB — به USDT
- **💵 ارز**: دلار (USD) و یورو (EUR) — به تومان
- **🪙 طلا**: طلای ۱۸ عیار و سکه امامی — به تومان

### تنظیم هشدار (۵ مرحله)

🔔 تنظیم هشدار ← بازار ← دارایی ← شرط (بالاتر/پایین‌تر) ← قیمت هدف ← ✅ تایید

نتیجه: وقتی شرط برقرار شد، بات پیام هشدار می‌فرستد (بررسی هر ۱۰ دقیقه).

### مشاهده و حذف هشدارها

📋 هشدارهای من ← لیست هشدارها با قیمت فعلی؛ برای حذف روی 🗑 حذف کنار هر هشدار بزنید.

## 📱 مثال استفاده

1. در تلگرام `@novax_price_bot` را باز کنید
2. `/start` را بزنید تا منوی ۴دکمه‌ای ظاهر شود
3. 💰 قیمت‌ها ← ₿ کریپتو را بزنید تا قیمت‌ها را ببینید
4. 🔔 تنظیم هشدار ← کریپتو ← BTC ← بالاتر از ← عدد هدف ← ✅ تایید
5. منتظر بمانید تا اعلان دریافت کنید!
