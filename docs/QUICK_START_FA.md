# شروع سریع - بات نواکس

## 🚀 بات آماده است!

بات نواکس الان در تلگرام فعال و آماده استفاده است.

### دسترسی به بات

در تلگرام به آدرس زیر بروید:

```
@novax_price_bot
```

یا از لینک زیر استفاده کنید:

```
https://t.me/novax_price_bot
```

## ⚡ دستورات سریع

### دریافت قیمت‌ها

```
/prices
```

نتیجه: قیمت لحظه‌ای دلار، طلا، سکه و تتر به تومان

### تنظیم هشدار

```
/alert USD 175000 above
```

نتیجه: وقتی دلار از ۱۷۵,۰۰۰ تومان بالاتر رفت، اعلان دریافت می‌کنید

### مشاهده هشدارها

```
/alerts
```

نتیجه: لیست همه هشدارهای فعال شما

## 📱 مثال استفاده

1. در تلگرام `@novax_price_bot` را باز کنید
2. دستور `/start` را بزنید
3. دستور `/prices` را بزنید تا قیمت‌ها را ببینید
4. هشدار تنظیم کنید: `/alert USD 175000 above`
5. منتظر بمانید تا اعلان دریافت کنید!
