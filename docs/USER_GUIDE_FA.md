# راهنمای کاربر بات نواکس

> راهنمای کامل و مفصل: [`docs/USER_GUIDE_COMPLETE_FA.md`](USER_GUIDE_COMPLETE_FA.md)

## معرفی

بات نواکس (@novax_price_bot) یک ربات تلگرام برای دریافت قیمت‌های لحظه‌ای و تنظیم هشدار قیمت است. بات **کاملاً منو‌محور (دکمه‌ای)** است؛ فقط `/start` را بزنید و بقیه‌ی کارها را از روی دکمه‌ها انجام دهید.

## شروع کار

در تلگرام به `@novax_price_bot` بروید و `/start` را ارسال کنید. منوی ۴دکمه‌ای ظاهر می‌شود:

```
┌─────────────────┬──────────────────┐
│ 💰 قیمت‌ها      │ 🔔 تنظیم هشدار  │
├─────────────────┼──────────────────┤
│ 📋 هشدارهای من │ ❓ راهنما        │
└─────────────────┴──────────────────┘
```

## 💰 مشاهده قیمت‌ها

روی **💰 قیمت‌ها** بزنید و بازار را انتخاب کنید:

- **₿ کریپتو**: BTC، ETH، SOL، BNB — منبع **Binance** — واحد **USDT**
- **💵 ارز**: دلار (USD)، یورو (EUR) — منبع **TGJU** — واحد **تومان**
- **🪙 طلا**: طلای ۱۸ عیار، سکه امامی — منبع **TGJU** — واحد **تومان**

## 🔔 تنظیم هشدار (فلوی ۵مرحله‌ای)

روی **🔔 تنظیم هشدار** بزنید و مرحله‌به‌مرحله پیش بروید:

1. **انتخاب بازار** (کریپتو / ارز / طلا)
2. **انتخاب دارایی** (مثلاً BTC)
3. **انتخاب شرط** (📈 بالاتر از / 📉 پایین‌تر از)
4. **وارد کردن قیمت هدف** (به‌صورت عدد، مثلاً `70000`)
5. **تایید** (روی ✅ تایید و ذخیره بزنید)

در هر مرحله می‌توانید با دکمه‌ی **🔙 بازگشت** به مرحله‌ی قبل برگردید. واحد کریپتو **USDT** و واحد ارز/طلا **تومان** است.

## 📋 مشاهده و حذف هشدارها

روی **📋 هشدارهای من** بزنید. لیست هشدارها همراه با **قیمت فعلی** هر دارایی نمایش داده می‌شود. برای حذف هر هشدار، روی دکمه‌ی **🗑 حذف** کنار آن بزنید.

## نکات مهم

1. **بررسی خودکار**: بات هر ۱۰ دقیقه یک‌بار قیمت‌ها را بررسی می‌کند (هشدار در بازه‌ی ۰ تا ۱۰ دقیقه پس از برقراری شرط ارسال می‌شود).
2. **منبع قیمت**: کریپتو از **Binance** و ارز/طلا از **TGJU**.
3. **واحدها**: کریپتو به **USDT**، ارز و طلا به **تومان**.

## وضعیت فنی

- **پلتفرم**: Cloudflare Workers
- **منبع قیمت**: Binance (کریپتو) + TGJU (ارز/طلا)
- **فرکانس بررسی**: هر ۱۰ دقیقه
- **ذخیره‌سازی**: Cloudflare KV
- **منطقه زمانی**: Asia/Tehran
