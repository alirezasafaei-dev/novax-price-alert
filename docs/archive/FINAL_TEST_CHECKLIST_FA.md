# چک‌لیست تست نهایی بات

## پیش‌نیازها

- [ ] Worker دیپلوی شده: `https://novax-telegram-relay.asdevelooper.workers.dev`
- [ ] Webhook تنظیم شده (از سیستم با اینترنت اجرا کنید: `./scripts/set-webhook.sh`)
- [ ] بررسی webhook: `curl -s "https://api.telegram.org/bot{TOKEN}/getWebhookInfo"`

## تست 1: شروع بات

**مراحل:**
1. در تلگرام جستجو کنید: `@novax_price_bot`
2. دکمه Start را بزنید

**انتظار:**
- پیام خوش‌آمد نمایش داده شود
- منوی دکمه‌ای 4 گزینه‌ای ظاهر شود:
  ```
  [ 💰 قیمت‌ها ]      [ 🔔 تنظیم هشدار ]
  [ 📋 هشدارهای من ] [ ❓ راهنما ]
  ```

## تست 5: تنظیم هشدار کریپتو (فلوی کامل)

**مراحل:**
1. کلیک روی `🔔 تنظیم هشدار`
2. کلیک روی `₿ کریپتو`
3. کلیک روی `BTC`
4. کلیک روی `بالاتر از`
5. تایپ کنید: `1`
6. کلیک روی `✅ تایید و ذخیره`

**انتظار:**
- در هر مرحله پیام مناسب نمایش داده شود
- دکمه‌های مربوط به هر مرحله ظاهر شوند
- در مرحله تایید، خلاصه هشدار نمایش داده شود
