# راهنمای کامل استفاده از بات هشدار قیمت نواکس

## معرفی

بات **نواکس** (@novax_price_bot) یک بات تلگرام برای دریافت قیمت‌های لحظه‌ای و تنظیم هشدار قیمت است.

## شروع استفاده

### 1. پیدا کردن بات

در تلگرام جستجو کنید: `@novax_price_bot`

### 2. شروع بات

دکمه **Start** را بزنید یا دستور `/start` را ارسال کنید.

### 3. منوی اصلی

بعد از شروع، منوی دکمه‌ای زیر را خواهید دید:

```
┌─────────────────┬──────────────────┐
│ 💰 قیمت‌ها      │ 🔔 تنظیم هشدار  │
├─────────────────┼──────────────────┤
│ 📋 هشدارهای من │ ❓ راهنما        │
└─────────────────┴──────────────────┘
```

## امکانات بات

### 💰 مشاهده قیمت‌ها

1. روی دکمه **💰 قیمت‌ها** کلیک کنید
2. بازار مورد نظر را انتخاب کنید:
   - **₿ کریپتو**: بیت‌کوین، اتریوم، سولانا، بایننس کوین
   - **💵 ارز**: دلار، یورو
   - **🪙 طلا**: طلای 18 عیار، سکه امامی

3. قیمت‌های لحظه‌ای نمایش داده می‌شود

**مثال خروجی:**

```
💰 قیمت‌های فعلی کریپتو

BTC: 67,240 USDT
ETH: 3,420 USDT
SOL: 165 USDT
BNB: 590 USDT

آخرین بروزرسانی: 14:35
```

### 🔔 تنظیم هشدار قیمت

#### مراحل تنظیم هشدار:

**مرحله 1: انتخاب بازار**

روی **🔔 تنظیم هشدار** کلیک کنید و بازار را انتخاب کنید:
- کریپتو
- ارز
- طلا

**مرحله 2: انتخاب دارایی**

دارایی مورد نظر را انتخاب کنید (مثلاً BTC)

**مرحله 3: انتخاب شرط**

شرط هشدار را مشخص کنید:
- **بالاتر از**: وقتی قیمت از مقدار هدف بالاتر رود
- **پایین‌تر از**: وقتی قیمت از مقدار هدف پایین‌تر رود

**مرحله 4: وارد کردن قیمت هدف**

قیمت هدف را به صورت عدد وارد کنید (مثلاً `70000`)

**مرحله 5: تایید**

خلاصه هشدار نمایش داده می‌شود:

```
لطفاً هشدار را تایید کن:

دارایی: BTC
شرط: بالاتر از 70,000
قیمت فعلی: 67,240

اگر قیمت BTC از 70,000 بالاتر رفت، به تو پیام می‌دهم.
```

روی **✅ تایید و ذخیره** کلیک کنید.

#### مثال کامل:

```
کاربر: [کلیک روی 🔔 تنظیم هشدار]
بات: کدام بازار؟
کاربر: [کلیک روی ₿ کریپتو]
بات: کدام دارایی؟
کاربر: [کلیک روی BTC]
بات: چه شرطی؟
کاربر: [کلیک روی بالاتر از]
بات: قیمت هدف چقدر باشد؟
کاربر: 70000
بات: [نمایش خلاصه و دکمه تایید]
کاربر: [کلیک روی ✅ تایید]
بات: ✅ هشدار ذخیره شد!
```

### 📋 مشاهده و حذف هشدارها

1. روی **📋 هشدارهای من** کلیک کنید
2. لیست هشدارهای فعال نمایش داده می‌شود
3. برای حذف هر هشدار، روی دکمه **🗑 حذف** کنار آن کلیک کنید

**مثال خروجی:**

```
📋 هشدارهای فعال شما:

1️⃣ BTC بالاتر از 70,000
   قیمت فعلی: 67,240
   [🗑 حذف]

2️⃣ ETH پایین‌تر از 3,000
   قیمت فعلی: 3,420
   [🗑 حذف]
```

### ❓ راهنما

روی **❓ راهنما** کلیک کنید تا راهنمای مختصر نمایش داده شود.

## دریافت هشدار

وقتی قیمت به شرط تعیین‌شده برسد، بات به صورت خودکار پیام هشدار ارسال می‌کند:

```
🔔 هشدار قیمت!

BTC به 70,500 رسید
شرط شما: بالاتر از 70,000

این هشدار غیرفعال شد.
```

**نکته مهم:** هر هشدار فقط یک بار ارسال می‌شود و بعد از آن غیرفعال می‌شود.

## بررسی هشدارها

بات هر **10 دقیقه** یک بار قیمت‌ها را بررسی می‌کند و اگر شرط هشدار برقرار باشد، پیام ارسال می‌کند.

## سوالات متداول

### چند هشدار می‌توانم تنظیم کنم؟

محدودیتی وجود ندارد. می‌توانید برای هر دارایی چندین هشدار با شرایط مختلف تنظیم کنید.

### آیا هشدار بعد از ارسال باز هم فعال است؟

خیر. هر هشدار فقط یک بار ارسال می‌شود. اگر می‌خواهید دوباره هشدار دریافت کنید، باید هشدار جدیدی تنظیم کنید.

### قیمت‌ها از کجا دریافت می‌شوند؟

- **کریپتو**: Binance API
- **ارز و طلا**: TGJU (بازار ایران)

### چرا هشدار من ارسال نشد؟

- مطمئن شوید قیمت واقعاً به شرط شما رسیده است
- بات هر 10 دقیقه بررسی می‌کند، پس ممکن است تا 10 دقیقه تاخیر داشته باشد
- هشدار را در لیست **📋 هشدارهای من** بررسی کنید که هنوز فعال باشد

### چطور هشدار را لغو کنم؟

از منوی **📋 هشدارهای من** روی دکمه **🗑 حذف** کنار هشدار مورد نظر کلیک کنید.
